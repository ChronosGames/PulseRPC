PulseRPC.Client.Unity 发布包结构
================================

/AOT/AOTSupport.cs                         - Unity/IL2CPP 框架类型保留入口
/Extensions/TaskExtensions.cs             - Unity 兼容任务辅助
/Plugins/*.dll                            - netstandard2.1 运行时完整依赖闭包
/Plugins/PulseRPC.Client.SourceGenerator.dll
                                            - 无 Workspaces/MEF 引用的 Unity 专用 analyzer
/Plugins/MemoryPack.Generator.dll         - MemoryPack Unity analyzer
/Samples~/BasicExample/                   - 可从 Package Manager 导入的 Hub/Receiver 示例
/Documentation~/dependency-closure.json  - DLL/analyzer 版本、引用与 SHA-256
/PulseRPC.Client.Unity.asmdef             - Unity assembly definition
/link.xml                                 - IL2CPP linker 保留配置
/package.json                             - 唯一 UPM manifest
/README.md                                - 当前使用说明

当前运行时传输为 TCP/KCP。包中不包含旧 WebSocket 客户端、手写 Generated 代理或
PulseClientFactory；Hub/Receiver 实现由 Roslyn Source Generator 针对消费项目生成。
